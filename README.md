## 🎵 Music Player

An interactive and responsive music player built using **HTML, CSS, and JavaScript**.  

This project allows users to play, pause, and switch between songs with a clean and modern user interface.

### ✨ Features
- ▶️ Play and pause functionality  
- ⏭️ Next and previous song controls  
- 🎧 Interactive audio player UI  
- 🎨 Responsive and modern design  
- 🔊 Smooth user experience with audio control  

### 💡 About
This project was built to strengthen frontend development skills, especially working with JavaScript audio handling, DOM manipulation, and UI design.
